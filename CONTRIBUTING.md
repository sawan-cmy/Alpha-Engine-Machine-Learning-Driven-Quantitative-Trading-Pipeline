# Contributing to AlphaEngine

First off, thank you for considering contributing to AlphaEngine! It's people like you that make AlphaEngine such a great tool.

## How to Contribute

1.  **Fork the repository** to your own GitHub account.
2.  **Clone the project** to your local machine:
    ```bash
    git clone https://github.com/your-username/AlphaEngine.git
    cd AlphaEngine
    ```
3.  **Create a branch** for your feature or bug fix:
    ```bash
    git checkout -b feature/my-new-feature
    ```
4.  **Make your changes** and test them thoroughly. Ensure your code follows the existing style and conventions.
5.  **Commit your changes** with a clear and descriptive commit message:
    ```bash
    git commit -m "Add some feature"
    ```
6.  **Push to the branch**:
    ```bash
    git push origin feature/my-new-feature
    ```
7.  **Submit a Pull Request** describing your changes and link any relevant issues.

## Reporting Bugs

If you find a bug, please create an issue with the following information:

*   A clear and descriptive title.
*   The exact steps to reproduce the problem.
*   What you expected to happen.
*   What actually happened instead.
*   Information about your environment (Python version, OS, etc.).

## Suggesting Enhancements

If you have an idea to improve AlphaEngine, please submit an issue explaining your proposed changes. Include detailed information about why this enhancement would be useful and how it might be implemented.

## Code Style

*   We try to adhere to standard Python web/data science conventions (PEP 8).
*   Use type hints wherever possible.
*   Ensure that any new feature includes appropriate tests and doesn't break existing functionality.

Thanks again for contributing!
